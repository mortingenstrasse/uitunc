Financial Tracking App UX/UI Case Study

Urbanist is a financial tracking application that enables users to monitor their expenses, income, and total balance in a clear and structured way. As a freelance designer, I led the end-to-end UX and UI design process. The primary goal was to present financial data in a simple, intuitive, and motivating format, helping users gain better control over their budgeting habits. The project included information architecture, user flows, dashboard design, and a clean visual system optimized for daily usability and clarity.
